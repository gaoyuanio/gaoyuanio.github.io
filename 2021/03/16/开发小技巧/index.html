提交事件不再重载页面

Mac显示隐藏文件命令：

显示隐藏文件

defaults write com.apple.finder AppleShowAllFiles Yes && killall Finder

不显示隐藏文件

defaults write com.apple.finder AppleShowAllFiles No && killall Finder
